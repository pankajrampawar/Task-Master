# styling

- Cards background color changes with priority the higher the priority darker will be the background
- Completed tasks have green background and are diplayed below incomplete task
- User can edit task by clicking on edit task icon
- user can delete task by clicking on delete task icon
- user can mark a task as completed by clicking on DONE button
- Tasks are stored in local storage in a string form and hence can be accesed even after reloading
